适配系统：macOS Sequoia 15−26

• 功能说明

• 一键屏蔽系统更新，无小红点、不自动下载

• 一键恢复更新，自动还原 hosts 和更新配置

• 纯原生命令实现，无第三方依赖，安全可控

• 使用说明

1. 前置准备

1. 克隆仓库或下载脚本文件

2. 终端执行一次性备份（仅需一次）：
   sudo cp /etc/hosts /etc/hosts.bak

3. 给脚本添加执行权限（脚本所在目录执行）：
   chmod +x 屏蔽更新.command 恢复更新.command

• 屏蔽更新

1. 双击运行 屏蔽更新.command

2. 按提示输入管理员密码

3. 执行完成后重启电脑生效

• 恢复更新

1. 双击运行 恢复更新.command

2. 按提示输入管理员密码

3. 执行完成后重启电脑生效
